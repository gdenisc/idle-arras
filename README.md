# Idle Arras

Thats a game where you farm shapes and collect them and then do even more shapes. You can join the [Discord server](https://discord.gg/ufeX6aR3NT) about it.

Feel free to help this game by making pull requests.

## How to start

1. npm i -D
2. go to the url in your terminal OR open dist/index.html (not recommended)
3. change the game files and reload webpage to apply changes
